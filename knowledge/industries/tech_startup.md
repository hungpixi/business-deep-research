# Tech Startup - Industry Template
# Focus: SaaS, AI Products, MVP-first approach

## Đặc thù ngành

### Giai đoạn phát triển startup (Startup Lifecycle)
1. **Ideation** → Validate problem-solution fit
2. **MVP/Seed** → Build MVP, get first users, raise seed
3. **Product-Market Fit** → Iterate until PMF, measure retention
4. **Growth/Series A** → Scale acquisition, hire team
5. **Scale/Series B+** → International expansion, new products
6. **Maturity/IPO** → Profitability, M&A, IPO

### Key Metrics By Stage

| Stage | Key Metric | Target |
|---|---|---|
| Pre-MVP | Problem validation interviews | >50 interviews |
| MVP | Active users, retention D7 | >40% D7 retention |
| PMF | NPS, MRR growth | NPS>40, MRR +15%/mo |
| Growth | ARR, LTV/CAC, NDR | LTV/CAC>3, NDR>100% |
| Scale | Rule of 40 (Growth% + Profit%) | >40 |

### Revenue Models cho Tech
- **SaaS Subscription**: Monthly/Annual plans, tiered pricing
- **Freemium**: Free tier + paid upgrades
- **Usage-based**: Pay per API call, storage, user
- **Marketplace**: Commission/take rate (10-30%)
- **API-as-a-Service**: B2B integrations

### Tech Stack Considerations
- Build vs Buy decisions
- Cloud infrastructure costs (AWS/GCP/Azure)
- Open source vs proprietary
- Technical debt management
- AI/ML infrastructure needs

### Regulatory & Legal (VN)
- ĐKKD: Đăng ký kinh doanh online qua https://dangkykinhdoanh.gov.vn
- Thuế: CIT 20%, VAT 8-10%, PIT cho founders
- IP: Đăng ký nhãn hiệu (NOIP), bảo hộ phần mềm
- Data: Luật An ninh mạng 2018, NĐ 13/2023 (PDPA)
- Startup incentives: Đề án 844, quỹ đầu tư khởi nghiệp

### Regulatory & Legal (US/Global)
- Incorporation: Delaware C-Corp (standard for VC)
- IP: Patent, trademark (USPTO)
- Compliance: GDPR (EU), CCPA (California), SOC2
- Export control: ITAR, EAR (for defense/dual-use tech)

### Funding Landscape
| Round | Typical Size | Valuation | Requirements |
|---|---|---|---|
| Pre-seed | $50K-$500K | $1-5M | Idea + team |
| Seed | $500K-$3M | $5-15M | MVP + traction |
| Series A | $3-15M | $15-50M | PMF + growth |
| Series B | $15-50M | $50-200M | Scale + revenue |

### VN Startup Ecosystem
- Accelerators: VIISA, Topica Founder Institute, Zone Startups
- VCs: Do Ventures, 500 Startups Vietnam, CyberAgent Capital, Genesia
- Events: Echelon, TechFest Vietnam
- Talent: Strong dev talent pool, competitive salaries rising

### AI Product Specifics (2026)
- LLM API costs declining rapidly
- Open source models (Llama, Mistral) viable alternatives
- Edge AI for mobile/IoT applications
- AI regulation emerging globally
- Differentiation through data moats, fine-tuning, and workflow integration
- Build AI-powered MVPs fast, validate with real users

### Go-to-Market Strategy cho AI Products
1. **PLG (Product-Led Growth)**: Free tier, viral loops
2. **Community-Led**: Open source + paid features
3. **Enterprise Sales**: Account-based marketing, demos
4. **API-first**: Developer community adoption
5. **Vertical SaaS**: Industry-specific solutions
